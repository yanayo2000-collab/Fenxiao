# Fenxiao MVP Scope

## 一期目标
- 跑通关系绑定
- 跑通收益事件入库
- 跑通三级奖励计算
- 提供后台基础查询和审计能力

## 当前已落地骨架
- Spring Boot + MySQL 项目初始化
- Flyway 建表脚本
- 核心领域实体：用户扩展、关系链、收益事件、奖励规则、奖励明细、风险事件、审计日志
- 基础前后台健康接口
- H2 测试环境

## 当前阶段结论
- 用户侧主链路已跑通：建档、绑定、首页、团队、奖励明细
- 运营侧主链路已跑通：登录、概览、奖励查询、风险查询、关系查询、风险处理、审计追踪
- 运营后台已完成一轮可用性打磨：逐条风险备注与确认、relation before / after 预览、relation 审计直查
- 当前系统已经达到“运营可用 MVP”标准，而不只是接口演示版

## 下一步开发顺序
1. Linky webhook 原始事件日志与 replay record 持久化
2. 提交整理与上线前 checklist
3. 运营使用文档与交接流程固化
4. 更细粒度权限与动作审批
5. 更深的审计详情与复核流
